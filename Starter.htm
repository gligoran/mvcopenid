This file is here as a work-around a Visual Studio 2010 bug, where it forgets which browser you set as the default.

To set your default browser just follow these steps:

  1. Right click on the Starte.htm file in your solution explorer.
  2. Select the "Browse With..." context menu option.
  3. The dialog you can select or add a browser. If you want Firefox in the list, click "add" and point to the firefox.exe filename.
  4. Click the "Set as Default" button to make this the default browser when you run any page on the site.
